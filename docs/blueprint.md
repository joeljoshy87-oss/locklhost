# **App Name**: Simple Journal

## Core Features:

- Entry Creation: Allow users to create new journal entries with text and optional multimedia attachments.
- Entry Display: Display journal entries in a chronological list or calendar view.
- Entry Categorization: Add functionality for users to categorize entries with tags, topics, or keywords to maintain organization and clarity.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to create a calming and reflective atmosphere.
- Background color: Very light blue (#F0F8FF) for a clean and unobtrusive backdrop.
- Accent color: Pale orange (#FFAD60) for interactive elements, to suggest warmth.
- Font: 'Literata' (serif) for readability and a classic, literary feel.
- Use simple, line-based icons to represent different entry categories and actions.
- Maintain a clean, single-column layout for readability, with a sidebar for navigation.
- Use subtle animations for transitions and interactive elements, like fading effects for new entries.
